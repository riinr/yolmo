# yolmo
Define cli tools with yml

# Features:

**Done:**

* Separete your command definition from command implementation

  * Shell: Maybe using [loki](/beshrkayali/loki)/[nim-prompt](/surf1nb1rd/nim-prompt)

  * GraphQl: GraphQl shell
